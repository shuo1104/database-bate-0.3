# 重启前端开发服务器

如果浏览器看不到更新，请重启前端：

## 方法1：在运行前端的终端
```bash
# 按 Ctrl+C 停止前端
# 然后重新启动
cd frontend_vue3
npm run dev
```

## 方法2：强制刷新浏览器
- Windows: `Ctrl + Shift + R` 或 `Ctrl + F5`
- Mac: `Cmd + Shift + R`

## 方法3：清除浏览器缓存
1. 按 F12 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"
